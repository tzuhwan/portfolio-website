Making Movie Recommendations Using MapReduce

This project is an implementation of the MapReduce Algorithm for parallel data processing. I applied MapReduce to create movie recommendations pipeline based on 571,073 ratings from 26,960 movies. The pipeline is as follows:

  • For every pair of movies A and B in the data set, find all people who rated both A and B;
  • Form ratings vectors for movies A and B;
  • Find the dot product between two vectors, the sum of each vector, the norm of each vector, and etc. to calculate the similarities between movie A and B based on Correlation, Regularized Correlation, Cosine Similarity and Jaccard Similarity metrics;
  • Receives a movie title input and outputs the top-n most similar movies to it.

The code is in a private GitHub repository and is available upon request. More information about the specs of this project can be found here.
